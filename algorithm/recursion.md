#### 递归算法
> 给定一个字典表示目录结构，打印其中所有的json文件名。字典表示文件夹，数组表示其中文件。

- 数据如下:
```
d = {
    "a": {
        "b": {
            "c": {
                "d": [
                    "e.json",
                    "f",
                    "g.json"
                ]
            }
        }
    },
    "h": {
        "i": [
            "j",
            "k.json",
            "l",
            "m",
            "n.json"
        ]
    },
    "o": {
        "p": {
            "q": []
        }
    },
    "r": ["s.json", "t.json"]
}
```
